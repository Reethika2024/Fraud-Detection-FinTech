# Credit Card Fraud Detection using XGBoost

**Objective:** Detect fraudulent credit card transactions using AI/ML techniques.

## Dataset
Download from Kaggle: [Credit Card Fraud Detection](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)  
Place `creditcard.csv` in the same folder as the notebook before running.

## Methodology
- Feature Engineering, Preprocessing, SMOTE, XGBoost, Evaluation

## How to Run
1. Clone this repo
2. Place the dataset in the folder
3. Open the notebook and run all cells
